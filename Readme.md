## AWS CI/CD Pipeline

### Class Notes

CI/CD is among core of devops (set of practice to build and deploy). CI/CD automates the building, testing and deployment phases in the software delievery model. For backend deployment we need to update cloud formation while for frontend we need to update s3 bucket.
